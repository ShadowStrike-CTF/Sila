# Sila

**Sila** (codename: Sierra) is part of the [Strategos](https://strategos.au)
Suite — professional forensic investigation platform.

Sila provides evidence management and immutable chain of custody across
the Strategos Suite.

This package is a namespace holder. Installing `sila` automatically
installs `sila-sierra`, the active development package.

## Install

pip install sila-sierra
